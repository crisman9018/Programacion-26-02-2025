portafolio

Aplicación de lista de tareas

Este proyecto consiste en crear una aplicación web que permita a los usuarios agregar, editar y eliminar tareas. La aplicación podría incluir características como la posibilidad de marcar tareas como completadas, asignar fechas de vencimiento y organizar las tareas en categorías. Este proyecto te ayudará a practicar el manejo de formularios, la manipulación del DOM y la utilización de bases de datos para almacenar la información de las tareas.

Sitio web de portafolio personal

En este proyecto, desarrollarías un sitio web donde puedas mostrar tus habilidades, proyectos y experiencias. El sitio web podría incluir secciones como una página de inicio, una biografía, una galería de proyectos y un formulario de contacto. Este proyecto te permitirá trabajar en el diseño y la estructura de un sitio web, así como en la integración de diferentes tecnologías como HTML, CSS y JavaScript.
